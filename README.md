# Strid3
Strid3 is an onchain engagement and reputation infrastructure that enables communities to track contributions, reward participation, and build long-term onchain reputation.

The project focuses on transforming offchain community activity into measurable onchain progress using XP, badges, and transparent reward distribution.

Communities today rely heavily on offchain tools such as chats, spreadsheets, and manual tracking to manage participation. These systems do not scale, lack transparency, and fail to build lasting reputation.

- Track meaningful contributions
- Reward participants fairly
- Prove participation onchain
- Improve retention and long-term engagement
